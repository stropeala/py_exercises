Encapsulation
Exercise 1: OOP Exercise
Create a class called Person.

The class should store:
• name (public attribute)
• age (private attribute)

Tasks:
1. Create a constructor that receives name and age.
2. Store age as a private attribute using __age.
3. Create a method get_age() that returns the age.

Goal: learn how private attributes protect data inside a class.
Exercise 2: OOP Exercise
Extend the Person class.

Instead of modifying the age directly, create a setter method.

Tasks:
1. Create a method set_age(new_age).
2. If the value is negative, raise a ValueError.
3. Otherwise update the private attribute.

Goal: understand how encapsulation controls how data is modified.
Exercise 3: OOP Exercise
Create a BankAccount class.

Attributes:
• owner (public)
• balance (private)

Tasks:
1. Constructor receives owner and initial balance.
2. Create deposit(amount).
3. Create withdraw(amount).
4. Create get_balance().

Goal: simulate a very small banking system.
Exercise 4: OOP Exercise
Improve the BankAccount class.

Withdrawals should not be allowed if the balance becomes negative.
Tasks:
1. Modify withdraw().
2. If withdrawal is larger than balance, raise an error.

Goal: enforce rules inside a class.
